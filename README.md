## Quick Start

```bash
# Install mux router
go get -u github.com/gorilla/mux
```

```bash
go build
./go_restapi
```

## Endpoints

### Get All Books

```bash
GET api/books
```

### Get Single Book

```bash
GET api/books/{id}
```

### Delete Book

```bash
DELETE api/books/{id}
```

### Create Book

```bash
POST api/books

```

### Update Book

```bash
PUT api/books/{id}
```
